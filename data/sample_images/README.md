# Images d'exemple / Sample images

Placez ici des images de PCB, microconnecteurs ou assemblages de biocapteurs
pour la demonstration.

Place PCB, microconnector or biosensor assembly images here for the demo.

## Formats acceptes / Accepted formats

- PNG, JPG/JPEG, BMP, TIFF, WebP

## Suggestions

- Images de PCB avec defauts visibles (brasures, composants manquants)
- Images de microconnecteurs (broches pliees, contamination)
- Images d'assemblages de biocapteurs

Vous pouvez telecharger des images d'exemple depuis des datasets publics :
- [PCB Defect Dataset (Kaggle)](https://www.kaggle.com/datasets)
- [MVTec Anomaly Detection](https://www.mvtec.com/company/research/datasets)
